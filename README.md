# Wallpaper Player

Windows에서 동영상을 배경화면으로 재생하는 프로그램입니다.

## 주요 기능

- ✅ 동영상 파일을 Windows 배경화면으로 재생
- ✅ 오디오 재생 지원
- ✅ 음소거 버튼 (배경화면에 표시)
- ✅ 설정 버튼으로 동영상 변경
- ✅ 설정 저장 (자동으로 마지막 선택한 동영상 재생)
- ✅ 작업표시줄을 가리지 않음

## 사용 방법

### 첫 실행
1. 프로그램을 처음 실행하면 동영상 선택 창이 나타납니다
2. 배경화면으로 사용할 동영상 파일을 선택합니다
3. 선택한 동영상이 배경화면에서 재생됩니다

### 버튼 사용
- **음소거 버튼** (오른쪽 하단): 클릭하여 음소거/음소거 해제
- **설정 버튼** (음소거 버튼 왼쪽): 클릭하여 동영상 변경

### 단축키
- `Q`: 프로그램 종료
- `F1`: 설정 창 열기

## 설정 파일

프로그램 폴더에 `wallpaper_config.json` 파일이 생성됩니다.
이 파일에는 다음 정보가 저장됩니다:
- 선택한 동영상 경로
- 음소거 상태
- 볼륨 설정

## 지원 동영상 형식

- MP4
- AVI
- MKV
- MOV
- WMV

## 시작 프로그램 등록

Windows 부팅 시 자동 실행을 원하면:

1. `setup_autostart.py` 실행
2. "1. 시작 프로그램에 등록" 선택

제거하려면 같은 방법으로 "2. 시작 프로그램에서 제거" 선택

## 빌드 방법 (개발자용)

```bash
# 아이콘 생성
python create_icons.py

# 실행 파일 빌드
pyinstaller build.spec --clean
```

빌드된 실행 파일은 `dist/WallpaperPlayer.exe`에 생성됩니다.

## 요구사항

- Windows 10/11
- Python 3.7+ (소스 코드 실행 시)
